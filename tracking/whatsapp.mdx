## Como Funciona o Rastreamento via WhatsApp

A maioria dos clientes não converte na primeira visita. Uma jornada típica se parece com isso:

1. O usuário clica em um anúncio do Facebook e acessa seu site (sessão web criada)
2. O usuário sai sem converter
3. Dias depois, o usuário entra em contato pelo WhatsApp
4. O usuário clica em um link enviado no WhatsApp e volta ao site
5. O usuário realiza a compra

Sem rastreamento entre canais, o passo 3 cria um perfil completamente separado. O Metrito resolve isso conectando conversas do WhatsApp a sessões web por meio de **identificadores de conexão**.

## Como a Conexão Acontece

Quando um usuário clica em um link enviado pelo WhatsApp com o parâmetro `wa_session` na URL, o pixel do Metrito captura esse parâmetro automaticamente e vincula a sessão web à conversa do WhatsApp.

```
https://seusite.com.br/checkout?wa_session=ID_DA_CONVERSA
```

A partir desse momento, o Metrito consegue conectar:
- O número de telefone do WhatsApp
- As sessões anteriores no site (pelo cookie do navegador)
- O e-mail preenchido em formulários futuros

O resultado é uma **jornada unificada**: o anúncio que trouxe o usuário pela primeira vez recebe crédito pela compra, mesmo que ela tenha acontecido dias depois via WhatsApp.

## Configurando Links com wa_session

Sempre que enviar links do WhatsApp para o seu site, adicione o parâmetro `wa_session`:

```
https://seusite.com.br/produto?wa_session=IDENTIFICADOR_UNICO
```

O identificador deve ser **único por conversa**. Pode ser o ID da conversa no seu sistema de atendimento, um hash gerado automaticamente, ou qualquer string que identifique aquela interação específica.

<Tip>
Se você usa ferramentas de automação de WhatsApp ou chatbots, configure-os para adicionar automaticamente o `wa_session` a todos os links enviados aos clientes.
</Tip>

## Exemplo de Jornada Conectada

```
Dia 1: Usuário clica em anúncio do Facebook
  → Visita o site com utm_source=facebook
  → Cookie do navegador é criado

Dia 3: Usuário envia mensagem no WhatsApp
  → Número de telefone é registrado

Dia 4: Atendente envia link com wa_session
  → Usuário clica e volta ao site
  → Metrito conecta: cookie do navegador ↔ wa_session ↔ telefone

Resultado: Anúncio do Dia 1 recebe atribuição pela compra do Dia 4
```

## Extensão Metrito Pixel Helper

O **Metrito Pixel Helper** é uma extensão do Chrome que ajuda a verificar e depurar o rastreamento em tempo real, incluindo sessões originadas do WhatsApp.

### O que Ela Faz

- Detecta o script do Metrito em qualquer site
- Exibe o **ID do Contêiner** da página atual
- Mostra os **dados do lead** salvos no navegador (ID, e-mail, telefone, nome)
- Captura e exibe eventos de rastreamento conforme são disparados
- Exibe um indicador visual quando o Metrito está ativo na página

### Como Instalar

1. Receba o arquivo da extensão com o time Metrito
2. Acesse `chrome://extensions/` no Chrome
3. Ative o **Modo do desenvolvedor** (canto superior direito)
4. Clique em **Carregar sem compactação** e selecione a pasta da extensão
5. O ícone do Metrito Pixel Helper aparece na barra de ferramentas

### Usando a Extensão

Com a extensão instalada, acesse qualquer página com o pixel do Metrito:

- Um pequeno badge **Metrito** aparece no canto inferior esquerdo da página confirmando a detecção
- Clique no ícone da extensão para ver:
  - Se o script foi detectado
  - O ID do contêiner
  - Dados do lead atual (ID, nome, e-mail, telefone)
  - Domínio rastreado

Quando eventos são disparados, uma notificação toast aparece no canto superior direito com o nome do evento.

## Boas Práticas

1. **Sempre adicione `wa_session` nos links enviados pelo WhatsApp** — é o principal elo de conexão entre o WhatsApp e a web
2. **Use o Pixel Helper para verificar** — confira se os dados do lead aparecem após envios de formulário
3. **Colete e-mail nas conversas do WhatsApp** — o e-mail é um identificador de alta confiança que conecta perfis mesmo sem `wa_session`
4. **Use o formato E.164 para telefone** — sempre inclua o código do país (ex: `+5511999999999`) para correspondência consistente
