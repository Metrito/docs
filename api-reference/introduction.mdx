## API de Eventos de Rastreamento

O principal endpoint para enviar eventos de rastreamento ao Metrito.

<Card title="Guia da API de Eventos" icon="terminal" href="/tracking/events-api">
  Documentação completa do endpoint `POST /v2/public/tracking/events`, incluindo schema da requisição, formato da resposta e exemplos de código.
</Card>

## URL Base

```
https://api.metrito.com
```

## Endpoint Principal

| Método | Caminho | Descrição |
|--------|---------|-----------|
| `POST` | `/v2/public/tracking/events` | Enviar um evento de rastreamento (público, sem autenticação) |

## Autenticação

O endpoint de rastreamento público (`/v2/public/tracking/events`) **não exige autenticação**. O campo `domain` no corpo da requisição identifica o contêiner de destino.
