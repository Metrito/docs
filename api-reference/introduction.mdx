## Tracking Events API

The primary API for sending tracking events to Metrito.

## Base URL

```
https://api.metrito.com
```

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/v2/public/tracking/events` | Send a tracking event (public, no auth) |
| `GET/POST` | `/v1/webhook/gateway?k=REF` | Receive webhooks from payment gateways |
| `POST` | `/v1/publish/tag` | Publish a tracking tag message |
| `POST` | `/v1/publish/zap` | Publish a WhatsApp/Zap message |

## Authentication

The public tracking endpoint (`/v2/public/tracking/events`) requires **no authentication**. The `domain` field in the request body identifies the target container.

Internal endpoints (`/v1/publish/*`) use bearer token authentication and are intended for server-to-server communication.
