## Por que os Parâmetros UTM são Importantes

Os parâmetros UTM informam ao Metrito qual campanha, conjunto de anúncios e criativo trouxe cada visitante ao seu site. Sem eles, o Metrito registra o tráfego mas não consegue atribuir as conversões aos anúncios específicos.

O Metrito utiliza o formato **nome|id** para capturar tanto o nome legível (para relatórios) quanto o ID técnico (para conciliação com as plataformas) em um único parâmetro.

## Meta Ads (Facebook e Instagram)

### Modelo de UTM Recomendado

Adicione este modelo nos seus anúncios do Meta Ads como Parâmetros de URL:

```
utm_source=facebook&utm_campaign={{campaign.name}}|{{campaign.id}}&utm_medium={{adset.name}}|{{adset.id}}&utm_content={{ad.name}}|{{ad.id}}&utm_term={{placement}}
```

### Como Configurar

<Steps>

<Step title="Abra o Gerenciador de Anúncios da Meta">
  Acesse o [Gerenciador de Anúncios](https://adsmanager.facebook.com) e selecione sua campanha.
</Step>

<Step title="Acesse o Rastreamento">
  Clique em **Editar** na campanha e role até a seção **Rastreamento**.
</Step>

<Step title="Adicione os Parâmetros de URL">
  No campo **Parâmetros de URL**, cole o modelo de UTM acima.
</Step>

<Step title="Salve">
  Clique em **Publicar**. Todos os anúncios desta campanha incluirão esses parâmetros automaticamente.
</Step>

</Steps>

<Tip>
Configure no **nível da campanha** para que todos os conjuntos de anúncios e anúncios herdem os parâmetros automaticamente. Você também pode configurar no nível do conjunto ou do anúncio para maior granularidade.
</Tip>

### O que os Parâmetros Dinâmicos da Meta Fazem

A Meta substitui os placeholders `{{chave_dupla}}` pelos valores reais no momento do clique:

| Parâmetro | Substituído por | Exemplo |
|-----------|-----------------|---------|
| `{{campaign.name}}` | Nome da campanha | `Promoção Verão Brasil` |
| `{{campaign.id}}` | ID da campanha | `123456789` |
| `{{adset.name}}` | Nome do conjunto de anúncios | `Mobile iOS 25-34` |
| `{{adset.id}}` | ID do conjunto | `987654321` |
| `{{ad.name}}` | Nome do anúncio | `Vídeo 50% Off` |
| `{{ad.id}}` | ID do anúncio | `456789123` |
| `{{placement}}` | Posicionamento | `feed`, `story`, `reel` |

### O que o Metrito Extrai

De uma URL como:

```
https://seusite.com.br/produtos?utm_source=facebook&utm_campaign=Promoção Verão|123456789&utm_medium=Mobile iOS|987654321&utm_content=Vídeo 50%25 Off|456789123&utm_term=feed&fbclid=IwAR1a2b3c
```

O Metrito interpreta e armazena:

| Campo | Valor |
|-------|-------|
| Fonte | `facebook` |
| Nome da campanha | `Promoção Verão` |
| ID da campanha | `123456789` |
| Nome do conjunto | `Mobile iOS` |
| ID do conjunto | `987654321` |
| Nome do anúncio | `Vídeo 50% Off` |
| ID do anúncio | `456789123` |
| Posicionamento | `feed` |

## Google Ads

### Modelo de Rastreamento Recomendado

No Google Ads, use um **modelo de acompanhamento** no nível da campanha:

```
{lpurl}?utm_source=google&utm_campaign={campaignname}|{campaignid}&utm_medium={adgroupname}|{adgroupid}&utm_content={creative}&utm_term={keyword}&gclid={gclid}
```

### Como Configurar

1. Acesse o **Google Ads** > selecione sua campanha
2. Clique em **Configurações** > **Opções de URL da campanha**
3. Cole o modelo de acompanhamento no campo **Modelo de acompanhamento**
4. Clique em **Salvar**

### Parâmetros Dinâmicos do Google Ads

| Parâmetro | Substituído por |
|-----------|-----------------|
| `{lpurl}` | URL da página de destino final |
| `{campaignname}` | Nome da campanha |
| `{campaignid}` | ID da campanha |
| `{adgroupname}` | Nome do grupo de anúncios |
| `{adgroupid}` | ID do grupo de anúncios |
| `{creative}` | ID do criativo/anúncio |
| `{keyword}` | Palavra-chave que ativou o anúncio |
| `{gclid}` | Google Click ID |

## TikTok Ads

### Modelo de UTM Recomendado

```
utm_source=tiktok&utm_campaign={{campaign.name}}|{{campaign_id}}&utm_medium={{adgroup.name}}|{{adgroup_id}}&utm_content={{ad.name}}|{{ad_id}}&ttclid={{ttclid}}
```

### Como Configurar

1. Acesse o **TikTok Ads Manager** > selecione sua campanha
2. Clique em **Editar** > role até **Rastreamento**
3. Em **Parâmetros de URL de Rastreamento**, cole o modelo
4. Clique em **Salvar**

## Configuração Personalizada de UTM

Se você já usa uma estrutura de UTM diferente, o Metrito pode ser configurado para interpretar o seu padrão existente. Entre em contato com o time de suporte para configurar o separador, a posição dos campos e o mapeamento personalizado.

## Boas Práticas

1. **Configure no nível da campanha** — aplica-se a todos os anúncios automaticamente
2. **Use o padrão `nome|id` recomendado** — garante tanto nomes legíveis quanto IDs técnicos
3. **Teste antes de escalar** — rode uma campanha de teste com orçamento pequeno e verifique a atribuição no Metrito
4. **Não mude o padrão no meio de uma campanha** — pode quebrar a atribuição histórica dos dados
5. **Use nomes consistentes** — nomes limpos tornam os relatórios muito mais úteis
6. **Sempre inclua os IDs de clique** — `fbclid`, `gclid`, `ttclid` são importantes para identificação entre canais
